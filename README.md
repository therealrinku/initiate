## Contribute to the Project

To run this project locally, follow these steps:

1. Clone the repository.
   ```bash
   git clone https://github.com/therealrinku/initiate.git

2. Install the dependencies.
   ```bash
   yarn install

3. Run the project.
   ```bash
   yarn start
